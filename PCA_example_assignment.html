With these variances, we see that principal component 1 is the largest and it composes the vast majority of variance. To determine which principal components to leave in the model, we will use Kaisers criterion, which retains only principal components that have a variance above 1. Here, we are left with only principal components 1 as all the others have a variance less than 1. Thus, we would leave only principal component 1 in the model.
